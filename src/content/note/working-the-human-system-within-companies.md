**00:38:11** — Working the Human System Within Companies

Michael describes the dinner-table observation that sparked PeopleWork: asking "how in the world did this person get that role?" — a conversation everyone has had over drinks with trusted people. The answer often comes down to being able to work the human system within a company. Someone who appears weak technically might be exceptional at communicating value to stakeholders. Using an engineering mindset — the same system automation mentality from DevOps and cloud engineering — Annie and Michael set out to create a system for managing these human dynamics intentionally.
